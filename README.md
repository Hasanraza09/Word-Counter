# Word Counter

## Description
This is a simple command-line tool built using TypeScript, Inquirer and Nodejs that allows users to input a sentence and get various statistics about it, such as word count, longest word, etc.

## Features
- Count the number of words in a sentence.
- Identify the longest word in the sentence.
- Display colorful output using Chalk library.
- Accept user input using Inquirer.

## Npx Command
You can use the following `npx` command:
```bash
npx hraza9-word-counter
